# Sitemap and Robots

`app/sitemap.ts` generates the sitemap. `app/robots.ts` generates robots.txt. Both use Next.js's `MetadataRoute` type.

## Sitemap

```typescript
// app/sitemap.ts
import type { MetadataRoute } from 'next'
import { client } from '@/tina/__generated__/client'

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const global = (await client.queries.global({ relativePath: 'global.json' })).data.global
  const pages = await client.queries.pageConnection()
  const posts = await client.queries.postConnection()

  const pageEntries =
    pages.data.pageConnection.edges
      ?.filter((e) => !e?.node?.seo?.noIndex && !e?.node?.draft)
      .map((edge) => ({
        url: `${global.siteUrl}/${edge?.node?._sys.filename === 'home' ? '' : edge?.node?._sys.filename}`,
        lastModified: edge?.node?._sys.lastModified
          ? new Date(edge.node._sys.lastModified)
          : new Date(),
        changeFrequency: 'weekly' as const,
      })) ?? []

  const postEntries =
    posts.data.postConnection.edges
      ?.filter((e) => !e?.node?.draft)
      .map((edge) => ({
        url: `${global.siteUrl}/blog/${edge?.node?._sys.filename}`,
        lastModified: edge?.node?.date ? new Date(edge.node.date) : new Date(),
        changeFrequency: 'monthly' as const,
      })) ?? []

  return [...pageEntries, ...postEntries]
}
```

Visit `/sitemap.xml` to verify.

## `MetadataRoute.Sitemap` shape

Each entry:

```typescript
{
  url: string,             // absolute URL
  lastModified?: Date | string,
  changeFrequency?: 'always' | 'hourly' | 'daily' | 'weekly' | 'monthly' | 'yearly' | 'never',
  priority?: number,       // 0 to 1
  alternates?: {
    languages?: Record<string, string>,  // for i18n
  },
}
```

## Filter drafts and `noIndex`

```typescript
const pageEntries = pages
  .filter((p) => !p.draft)              // exclude drafts
  .filter((p) => !p.seo?.noIndex)       // exclude noIndex
  .map(/* ... */)
```

Both filters are critical — drafts publicly indexed by Google = SEO disaster.

## Pagination for large sitemaps

Sitemaps over 50,000 URLs need to be split. Use sitemap index:

```typescript
// app/sitemap.ts (index)
export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  // For up to 50,000 URLs — single file
  return /* ... */
}

// For more, split into multiple sitemap files:
// app/sitemap-pages.xml/route.ts
// app/sitemap-posts.xml/route.ts
// app/sitemap.xml/route.ts (index pointing to the above)
```

For most sites, the single-file pattern is enough.

## Robots

```typescript
// app/robots.ts
import type { MetadataRoute } from 'next'

export default function robots(): MetadataRoute.Robots {
  return {
    rules: {
      userAgent: '*',
      allow: '/',
      disallow: ['/admin/', '/api/'],
    },
    sitemap: `${process.env.NEXT_PUBLIC_SITE_URL}/sitemap.xml`,
  }
}
```

`disallow: ['/admin/', '/api/']` — keeps admin and API routes out of search results.

## Per-user-agent rules

```typescript
return {
  rules: [
    {
      userAgent: 'Googlebot',
      allow: '/',
      disallow: ['/admin/', '/api/'],
    },
    {
      userAgent: ['GPTBot', 'CCBot'],  // block AI crawlers if you want
      disallow: '/',
    },
    {
      userAgent: '*',
      allow: '/',
      disallow: ['/admin/', '/api/'],
    },
  ],
  sitemap: `${process.env.NEXT_PUBLIC_SITE_URL}/sitemap.xml`,
}
```

## Verifying

After deploy:

```bash
curl https://your-site.com/sitemap.xml
curl https://your-site.com/robots.txt
```

Should return valid XML and text files. Submit `sitemap.xml` URL to Google Search Console.

## Caching

Sitemap and robots files are statically generated by default (cached for the build). For very large sitemaps that change frequently, set `export const revalidate = 3600`.

## Common mistakes

| Mistake | Effect | Fix |
|---|---|---|
| Forgot to filter drafts | Drafts indexed by Google | Filter `draft: true` |
| Forgot to filter `noIndex` | Pages with `seo.noIndex` indexed | Filter accordingly |
| Used relative URLs | Sitemap invalid | Always absolute |
| Wrong `lastModified` format | Validation error | Use Date or ISO string |
| Forgot `sitemap.xml` link in robots | Search engines may miss it | Include in robots |
| Disallowed `/api/tina` | Self-hosted CMS doesn't work | Be specific: `/admin/` and `/api/<your-routes>/` only |
