# audit-skill-by-derailment

Hardening an existing skill by running a real-task subagent derailment pass, classifying trace friction, and editing the skill files.

**Category:** productivity

## Install

Install this skill individually:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur-secondary/skills/audit-skill-by-derailment
```

Or install the full pack:

```bash
npx -y skills add -y -g yigitkonur/skills-by-yigitkonur-secondary
```
